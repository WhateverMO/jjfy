模型算法部分的输入输出

输入：
questionnaire_1_counts：问卷一输入，结构同原来

questionnaire_2_counts：问卷二输入，结构同原来

q1_section_1：一二级指标的模块划分，例子：[3,3,2,2]

q1_section_2：三级指标的模块划分，例子：[2,2,1,2,2,1,2]

l1_names：一级指标名称，结构为：["A", "B", "C"]

l2_names：二级指标名称，结构为：[["A1", "A2", "A3"], ["B1", "B2"], ["C1", "C2"]]

l3_names：三级指标名称，结构为：[["A11", "A12"], ["A21", "A22"], ["A31"], ["B11", "B12"], ["B21", "B22"], ["C11"], ["C21", "C22"],]

q2_section：问卷二的模块划分（模块定义：所有关于某一个三级指标的题目属于一个模块，结构为：[[1,2,3],[4,6],[5,7],...] ；含义：第1、2、3题是关于第一个三级指标的题目，第4、6是关于第二个三级指标的题目，第5、7是关于第三个三级指标的题目

method = "bwm"：选择的方法

注：其中A代表一级指标，A1代表一级指标的第一个二级指标，A11代表一级指标的第一个二级指标的第一个三级指标

输出：
字典，同原来
